# Employee Work Monitoring System

A cybersecurity-based employee monitoring system with:

- Host/Admin Login
- User Login
- Secure Notes
- Activity Logs
- Dashboard Monitoring
- MySQL Database
- Node.js Backend

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Run Project

Backend:
npm install
npm run dev

Frontend:
Open index.html
